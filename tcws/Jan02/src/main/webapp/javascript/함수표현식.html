함수표현식

함수표현식이란 함수인데 이름이 없는 함수를 말한다. 함수는 처음부터 메모리를 차지하고 있는 거와 달리 함수표현식을 메모리를 차지하고 있지 않고 호출될 때 비로소 메모리가 확보되고 실행 후 메모리가 사라진다. 보통은 이벤트 핸들러나 콜백함수를 만들 때 주로 사용된다. 가볍게 만들어서 사용하고 버리는 함수를 말한다. 선언 방법은 변수명 = function(매개변수){ } 형태이다. 함수의 이름이 없다. 왼쪽 변수에 함수의 주소를 전달하고 나중에 변수를 이용해 함수를 호출할 수 있다. 콜백함수는 함수를 만드는 사용자와 호출자가 다른 함수를 말 한다. 함수를 만들어서 호출자에게 함수의 주소를 전달해주면 호출자는 적당한 시점에 콜백함수를 호출한다. 이벤트 핸들러를 구축하거나, 시스템의 일부분만을 별도의 코드로 작성해야 할 때 주로 사용한다. 만드는 사람은 전체 시스템의 내부구조는 몰라도 정해진 범위 안에서 함수를 만들어 전달해준다.
