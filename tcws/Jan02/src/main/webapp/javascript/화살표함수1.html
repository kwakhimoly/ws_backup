화살표함수

화살표함수는 함수가 비교적 짧고 간단해서 별도의 함수를 만들기에는 애매하고, 굳이 메모리를 계속 잡고 있을 필요도 없고, 전체 시스템에서 일부만을 담당할 때 이름도 없이 간단하게 만들어서 쓰고 버릴 수 있다. (일회용품 같은 느낌)
-> 함수표현식의 업그레이드 버전
단점 : 이 표현식은 this 인자를 사용 못함. 함수표현식은 사용 가능하지만 화살표함수는 내부에서 this를 쓸 수 없음. 

	일반함수

	function add(a,b){
		return a+b;
	}

	함수표현식
	add = function(a,b){
		return a+b;
	}

	화살표 함수 (람다)
	add = (a,b) => {
		return a+b
	}
